# Frontend AI Engineering Capstone

This repository contains my capstone project for the Frontend AI Engineering track.

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Git
- GitHub

## Goals

- Learn AI-assisted development
- Build frontend projects
- Practice Git workflow

## Contributing

Thanks for your interest in contributing! This project is in early development, so the setup is intentionally minimal for now.

### Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- [Git](https://git-scm.com/)
- A GitHub account

### Local setup

1. Fork this repository on GitHub, then clone your fork:

   ```bash
   git clone https://github.com/<your-username>/frontend-ai-capstone.git
   cd frontend-ai-capstone
   ```

2. Create a feature branch for your work:

   ```bash
   git checkout -b feat/short-description
   ```

3. As frontend projects are added to the repo, each folder will include its own run instructions (for example, `npm install` and `npm run dev`). Check the project folder's README or `package.json` scripts before opening a pull request.

### Coding conventions

- Use meaningful variable names
- Write modular code with small, focused functions
- Use ES6+ syntax
- Follow [Conventional Commits](https://www.conventionalcommits.org/) for commit messages (for example, `feat: add landing page layout`)

### Submitting a pull request

1. Push your branch to your fork and open a PR against `main`
2. Describe what you changed and why in the PR description
3. Keep PRs focused—one feature or fix per pull request
4. Respond to review feedback and update the branch as needed

Questions or ideas? Open an issue before starting large changes so we can align on scope.
